# 📝 환경 변수 입력 가이드 (5분 완성)

## Render Dashboard에서 환경 변수 입력하기

### 접속 방법
1. https://dashboard.render.com 로그인
2. 서비스 선택 → **Environment** 탭 클릭
3. **Add Environment Variable** 클릭

---

## 입력할 환경 변수 (총 12개)

### 데이터베이스 (6개)

| Key | Value | 복사 위치 |
|-----|-------|-----------|
| `DB_HOST` | `[호스트].aivencloud.com` | Aiven 콘솔 |
| `DB_PORT` | `26163` (예시) | Aiven 콘솔 |
| `DB_USER` | `avnadmin` | Aiven 콘솔 |
| `DB_PASSWORD` | `[비밀번호]` | Aiven 콘솔 |
| `DB_NAME` | `defaultdb` | Aiven 콘솔 |
| `DB_SSL` | `true` | 직접 입력 |

### AI 및 이미지 (4개)

| Key | Value | 복사 위치 |
|-----|-------|-----------|
| `GEMINI_API_KEY` | `[API 키]` | Google AI Studio |
| `CLOUDINARY_CLOUD_NAME` | `[클라우드명]` | Cloudinary 대시보드 |
| `CLOUDINARY_API_KEY` | `[API 키]` | Cloudinary 대시보드 |
| `CLOUDINARY_API_SECRET` | `[시크릿]` | Cloudinary 대시보드 |

### 자동 설정 (2개) - 입력 불필요

| Key | Value | 비고 |
|-----|-------|------|
| `NODE_ENV` | `production` | render.yaml 자동 |
| `PORT` | `10000` | render.yaml 자동 |

---

## 완료 후

1. **Save Changes** 클릭
2. 자동으로 재배포 시작됨
3. 빌드 완료까지 3~5분 대기
4. 접속 테스트:
   - 환자 포털: `/patient`
   - 관리자 대시보드: `/admin`
