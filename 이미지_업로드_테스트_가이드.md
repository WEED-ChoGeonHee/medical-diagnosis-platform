# 이미지 업로드 테스트 가이드

## 테스트 시나리오

### 1. 회원가입/로그인
```
URL: https://medical-diagnosis-platform.onrender.com/patient/register
- 이메일: test@test.com
- 비밀번호: test123456
- 이름: 테스트환자
- 전화번호: 01012345678
```

### 2. 진단 요청 생성
```
- 환자 이름: 테스트환자
- 증상 종류: 여드름/뾰루지
- 피부 타입: 지성
- 증상 설명: 얼굴 전체에 붉은 여드름이 생겼습니다.
- 이미지: 1-5개 업로드
```

### 3. 확인 사항

#### ✅ 업로드 확인
- [ ] 파일 선택 버튼 작동
- [ ] 이미지 미리보기 표시
- [ ] 업로드 진행 중 로딩 표시
- [ ] 업로드 완료 후 진단 상세 페이지 이동

#### ✅ 이미지 표시 확인
- [ ] 진단 상세 페이지에서 이미지 표시
- [ ] Cloudinary URL로 이미지 로드
- [ ] 여러 이미지 갤러리 형태로 표시
- [ ] 이미지 클릭 시 확대 (있는 경우)

#### ✅ 의사 대시보드 확인
```
URL: https://medical-diagnosis-platform.onrender.com/admin
- 환자 목록에서 이미지 있는 진단 확인
- 진단 상세에서 이미지 정상 표시
```

## 문제 해결

### 이미지가 안 보이는 경우
1. 브라우저 개발자 도구 (F12) 열기
2. Network 탭에서 이미지 요청 확인
3. Console 탭에서 에러 확인

### Cloudinary 환경변수 확인
```
https://medical-diagnosis-platform.onrender.com/api/debug/db
```
- CLOUDINARY_CLOUD_NAME: ✅ set
- CLOUDINARY_API_KEY: ✅ set
- CLOUDINARY_API_SECRET: ✅ set

## 예상 이미지 URL 형식
```
https://res.cloudinary.com/YOUR_CLOUD_NAME/image/upload/v1234567890/medical-diagnosis/abcdef.jpg
```
